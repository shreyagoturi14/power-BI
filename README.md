# Power BI Dashboard Projects

## Overview

This repository contains my collection of **Power BI Dashboard Projects** built to analyze real-world business data and transform it into meaningful insights using interactive visualizations.

These projects focus on data analysis, dashboard design, KPI tracking, and business intelligence across multiple domains such as E-Commerce, HR Analytics, and Retail Sales.

---

## Projects Included

### 1. Madhav E-Commerce Sales Dashboard
Analysis of sales performance, profit trends, customer behavior, payment methods, and state-wise sales distribution.

### 2. HR Analytics Dashboard
Employee attrition analysis including workforce demographics, salary distribution, job roles, and retention insights.

### 3. Blinkit Sales Dashboard
Retail sales analysis covering outlet performance, item categories, outlet size, outlet location, and customer buying patterns.
