# Blinkit Sales Dashboard

## Overview

This Power BI project presents an interactive **Blinkit Sales Dashboard** designed to analyze sales performance, outlet establishment trends, product categories, outlet sizes, and customer buying patterns.

The dashboard helps businesses understand sales distribution, outlet performance, item visibility, and customer preferences through meaningful visual insights.

---

## Key Features

- Total Sales Performance Analysis
- Average Sales Tracking
- Number of Items Sold
- Average Customer Rating
- Outlet Establishment Trend Analysis
- Fat Content-Based Sales Distribution
- Item Type Performance Analysis
- Outlet Size Comparison
- Outlet Location Tier Analysis
- Outlet Type Performance Breakdown
- Dynamic Filter Panel for Better Analysis

## Sales Analysis

### Outlet Establishment

- Sales growth significantly increased after 2011
- Highest outlet performance observed around **2018–2022**
- Peak sales reached approximately **$131K**

### Fat Content Analysis

- **Low Fat** products contribute the highest sales
- Regular products also show strong sales contribution

### Item Type Performance

Top performing item categories include:

- Fruits and Vegetables
- Snack Foods
- Household
- Frozen Foods
- Dairy

These categories generate the highest revenue contribution.

### Outlet Size Analysis

- **Medium-sized outlets** generate the highest sales
- Followed by Small and High-sized outlets

### Outlet Location Analysis

- **Tier 2 locations** contribute the highest sales
- Tier 3 follows next
- Tier 1 shows comparatively lower sales

### Outlet Type Performance

Top performing outlet type:

- **Supermarket Type 1**

This outlet type shows the highest sales, item count, and strong customer ratings.
